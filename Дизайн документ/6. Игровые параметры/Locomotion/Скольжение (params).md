# 1. BaseSlideSpeed

**Type:** `float`  
**Value:** `4.0f`
**Описание:**  
Базовое значение скорости скольжения, принимающее участие в расчёте целевой скорости скольжения.

Это значение задаёт **начальную скорость**, которая далее модифицируется параметрами:

- наклона поверхности;
- трения поверхности;
- ввода игрока.

Используется как основа для расчёта итоговой скорости скольжения.
# 2. InputMultiplier
## 2.1 BackMultiplier

**Type:** `float`  
**Value:** `0.6f`
**Описание:**  
Множитель скорости, применяемый при вводе движения **против направления скольжения**.

Позволяет игроку **замедлять скольжение**.
## 2.2 ForwardMultiplier

**Type:** `float`  
**Value:** `1.4f`
**Описание:**  
Множитель скорости, применяемый при вводе движения **по направлению скольжения**.

Позволяет игроку **ускорять движение вниз по склону**.

# 3. SlopeMultiplier
## 3.1 MinSlopeMultiplier

**Type:** `float`  
**Value:** `1.0f`
**Описание:**  
Минимальный множитель наклона поверхности.

Применяется при скольжении на поверхностях, угол которых находится у [[Поверхности (params)#SlideSurfaceAngles|нижней границы диапазона скольжения]].

## 3.2 MaxSlopeMultiplier

**Type:** `float`  
**Value:** `1.8f`
**Описание:**  
Максимальный множитель наклона поверхности.

Применяется при скольжении на поверхностях, угол которых приближается к [[Поверхности (params)#SlideSurfaceAngles|верхней границе диапазона скольжения]].
