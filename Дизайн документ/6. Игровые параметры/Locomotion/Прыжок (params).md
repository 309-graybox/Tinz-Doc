# 1. JumpSurfaceIntersectionMask
**Type:** `int`
**Value:** `~1`
**Описание:** Маска, которая определяет возможность персонажа прыгнуть оттолкнувшись от отмеченной поверхности.
# 2. JumpSpeed

**Type:** `float`
**Value:** `?` #todo 
**Описание:** Начальная вертикальная скорость которая задается персонажу во время прыжка.

# 3. Gravity

**Type:** `float`
**Value:** `-9.81f`
**Описание:** Ускорение которое применяется к [[#1. JumpSpeed|JumpSpeed]], думаю лучшим вариантом будет забрать величину из параметра Gravity в разделе Physics движка.

# 4. AdaptiveJumpDamping

**Type:** `float`
**Value:** `0.6f`
**Описание:** Процент на который должна уменьшится текущая вертикальная скорость персонажа, для реализации механики [[Прыжок#Адаптивный прыжок|адаптивного прыжка]]. 

# 5. AdaptiveJumpTreshold

**Type:** `float`
**Value:** `0.05`
Пороговое значение вертикальной скорости (в процентах от [[#2. JumpSpeed|начальной скорости прыжка]]),  
ниже которого [[Прыжок#Адаптивный прыжок|адаптивный прыжок]] перестаёт применяться.

# 6. JumpInputBufferTimer

**Type:** `float`
**Value:** `0.15f`
**Описание:** Таймер времени буферизации ввода прыжка.

# 7. CoyoteTime

**Type:** `float`
**Valve:** `0.1f`
**Описание:** Таймер определяющий время, в течении которого персонаж может совершить прыжок не имея контакта с поверхностью. 

# 8. ForwardPushLength

**Type:** `float`
**Valve:** `?` #todo 
**Описание:** Расстояние на которое система может подвинуть персонажа для продолжения прыжка.
