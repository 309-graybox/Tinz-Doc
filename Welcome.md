# Навигация с помощью тегов

## Справочные теги:

#welcome - тег ведущий только на эту страницу

#help - страницы помеченные этим тегом содержат справочную информацию по хранилищу 

#todo - помечены места в которых документ не дописан
## Организационные теги:

#new - этим тегом помечаются заметки которые были изменены в последнем релизе, на них стоит обратить внимание в первую очередь

#report - этим тегом помечаются заметки с описанием важных сообщений к разработчикам (Баги, Проблемы, Срочные Объявления)

#work-progress  - тег ведущий на страницу с текущим прогрессом работы

#commented - этим тегом разработчики могу пометить заметку с комментариями разработчиков

# Как комментировать документы

Чтобы прокомментировать что-то в тексте заметки нужно обернуть текст в `%%` 
пример: `%%текст комментария%%` 

> **Внимание!**
> Текст обернутый в комментарий, не отображается в режиме просмотра

# Полезности

Можно просматривать содержимое внутренних ссылок без перехода по ним.
Для этого нужно зажать `Ctrl` и навести на [[Welcome#Полезности|внутреннюю ссылку]] курсор мыши.
